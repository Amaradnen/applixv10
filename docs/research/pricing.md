# Pricing & Subscription Models - Competitor Research

## Research Date
January 21, 2026

## Top Competitors (SaaS Pricing Pages)

### 1. Stripe (Benchmark for Clarity)
**Website:** https://stripe.com/pricing
**What They Do Well:**
- **Transparent Pricing:** Clear percentage + fixed fee
- **No Hidden Costs:** Upfront about all fees
- **Calculator:** Interactive pricing calculator
- **Feature Comparison:** Clear table of features per plan
- **Volume Discounts:** Clear thresholds for enterprise

### 2. Linear (Modern SaaS Pricing)
**Website:** https://linear.app/pricing
**What They Do Well:**
- **Simple Tiers:** Free, Standard, Plus, Enterprise
- **Per-Seat Pricing:** Clear $$/user/month
- **Annual Discount:** Save 20% when billed annually
- **Feature Highlights:** Key differentiators per plan
- **No Tricks:** Honest, straightforward

### 3. Vercel (Developer-Focused)
**Website:** https://vercel.com/pricing
**What They Do Well:**
- **Generous Free Tier:** Real value for individuals
- **Clear Limits:** Bandwidth, build minutes, etc.
- **Usage-Based:** Pay for what you use
- **Team Features:** Collaboration at higher tiers
- **Enterprise:** Custom for large orgs

### 4. Notion (Freemium Done Right)
**Website:** https://notion.so/pricing
**What They Do Well:**
- **Free Forever:** No credit card required
- **Clear Upgrade Path:** When to upgrade is obvious
- **Plan Comparison:** Side-by-side table
- **Add-Ons:** AI features priced separately
- **Flexible Billing:** Monthly or annual

## Common Pricing Patterns

### Tier Structure (Most Common)
```
Tier 1: Free/Starter
- Limited features
- Good for individuals/testing
- Credit card optional

Tier 2: Pro/Plus
- All core features
- For professionals/small teams
- $10-50/month

Tier 3: Business/Team
- Advanced features
- For teams/companies
- $50-200/month

Tier 4: Enterprise
- Custom everything
- For large organizations
- $500+/month or custom
```

### Billing Options
- **Monthly:** Higher price, cancel anytime
- **Annual:** Save 15-25%, billed upfront
- **Usage-Based:** Pay for what you use (Stripe model)
- **Per-Seat:** Price per user (Linear model)

## Gaps & Opportunities

### What Competitors Miss:
1. **No Plan Configurator:** Can't customize which features you need
2. **Hidden Add-Ons:** Extra costs discovered later
3. **Confusing Limits:** "Up to 10,000 API calls" - what happens after?
4. **No Savings Calculator:** How much do I save annually?
5. **Generic Plans:** Everyone gets same features, no flexibility
6. **Poor Mobile View:** Pricing tables don't work on mobile
